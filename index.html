Merhaba Barış! Cloudflare'de domainin varsa, HTML/CSS/JavaScript ile statik bir site kurabiliriz. Cloudflare Pages tam bunun için ideal - ücretsiz, hızlı ve doğrudan GitHub'dan deploy edebiliyorsun.

Yapılacak Adımlar:

1. Basit HTML Site Tasarımı

Sana modern, responsive bir site kodu yazayım. Hangi tür bir site istiyorsun?
  • Kişisel portfolyo (CV, projeler, iletişim)
  • Landing page (Tek sayfalık tanıtım sitesi)
  • Kurumsal site (Hakkında, hizmetler, iletişim)
  • Blog tarzı (Yazılar, kategoriler)

2. Cloudflare Pages'a Yükleme

  • GitHub reposu oluşturup kodu oraya atacağız
  • Cloudflare Pages ile bağlayıp otomatik deploy edeceğiz
  • Domainini buna yönlendireceğiz

Hemen Başlayalım mı?

Bana şunu söyle:
  1. Site türü ne olsun? (Yukarıdaki seçeneklerden)
  2. Site başlığı/adı ne olsun? (Örn: "Barış'ın Portfolyosu")
  3. Hangi renk teması? (Koyu/açık, belirli renkler?)
  4. İçerikte neler olsun? (Hakkında metni, iletişim bilgileri vb.)
Bu bilgileri verirsen, sana özel hazırlanmış, modern bir site kodu yazayım ve Cloudflare Pages'a nasıl yükleyeceğini adım adım anlatayım.

Barış, Fibrastone için profesyonel bir site hazırladım! Artstone'dan esinlenerek modern, koyu temalı şık bir tasarım yaptım.

Site Özellikleri:

🎨 Tasarım

  • Koyu hero section (logonun kırmızısıyla uyumlu)
  • Modern, minimalist yapı
  • Responsive (mobil uyumlu)
  • Smooth scroll navigasyon

📄 Bölümler

  1. Hero - Büyük başlık ve CTA butonları
  2. Özellikler - 4 ana avantaj kartı (hafif, yangın dayanımı, tasarım, montaj)
  3. Ürünler - 6 kategori (duvar panelleri, cephe, söve, kolon, saksı, özel üretim)
  4. Uygulamalar - 6 farklı sektör (otel, ticari, konut, kültürel, eğitim, sağlık)
  5. Teknik Bilgiler - Detaylı spec listesi
  6. İletişim - Form ve iletişim bilgileri
  7. Footer - Linkler ve sosyal medya

⚡ Teknik

  • Tek HTML dosyası (kolay deploy)
  • CSS içinde (harici dosya yok)
  • JavaScript smooth scroll için
  • Font: Montserrat (Google Fonts)

Sıradaki Adımlar:

1. Logoları Ekle

Logolarını (fibrastone1.png ve fibrastoneBEYAZ.png) siteye eklemek için:
  • Beyaz logo → Hero section ve dark arka planlı yerler
  • Siyah/kırmızı logo → Navbar ve açık arka planlı yerler

2. Ürün Fotoğrafları

Her ürün kartındaki placeholder'ları gerçek ürün fotoğraflarınla değiştirebilirsin.

3. Cloudflare Pages'a Yükleme

Hazır mısın? Şimdi Cloudflare Pages'a nasıl yükleyeceğini adım adım anlatayım:
GitHub + Cloudflare Pages Deploy:
  1. GitHub Repo Oluştur
    • GitHub'da yeni repo aç (örn: fibrastone-website)
    • Bu HTML dosyasını index.html olarak yükle
  2. Cloudflare Pages
    • Cloudflare dashboard → Pages
    • "Create a project" → GitHub'ı bağla
    • Repoyu seç, framework preset: None
    • Build command: boş bırak
    • Deploy!
  3. Domain Bağlama
    • Custom domains → Domainini ekle
    • DNS ayarlarını Cloudflare'den yap
Veya basit yol: Cloudflare Pages'te "Direct Upload" ile direkt bu HTML dosyasını yükleyebilirsin.
